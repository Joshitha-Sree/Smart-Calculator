# 📖 About the Project

Welcome to my **Smart Calculator** project!

This is the **first software engineering project** in my B.Tech Electronics and Communication Engineering (ECE) journey. While the calculator performs basic arithmetic operations, the real objective of this project is to build a strong foundation in **Python programming**, **Git**, **GitHub**, and professional software development practices.

Rather than simply writing a calculator, I wanted to learn **how real software projects are planned, structured, documented, version controlled, and continuously improved.**

This repository represents the beginning of my long-term engineering portfolio.
